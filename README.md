 Netflix Clone

This project is a **Netflix landing page clone** created using **HTML and CSS**. The goal of this project was to practice front-end web development and understand how modern websites are structured and styled. The layout, sections, and design are inspired by the official Netflix homepage. Through this project, I learned how to use **Flexbox, CSS Grid, responsive design, and proper HTML structure** to build a visually appealing webpage.

## Project Highlights

* Built using **pure HTML and CSS**
* Responsive layout using **Flexbox and CSS Grid**
* Netflix-style **hero section with background image**
* **Navigation bar** with language and sign-in buttons
* **Email signup section** similar to Netflix
* Multiple **content sections with images and videos**
* **FAQ section** with interactive hover effects
* **Footer section** with useful links
* Organized **project folder structure**

##  Purpose of the Project

* To improve my **front-end development skills**
* To practice building **real-world UI layouts**
* To understand **responsive web design**
